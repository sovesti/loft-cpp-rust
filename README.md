# loft-ccp-rust
C/C++ Tools

## Build
You will need installed [Rust](https://www.rust-lang.org/tools/install) to build this project.
To build, execute
```
cd loft-cpp-rust
cargo build -r
```
To run:
```
cd target/release
loft-cpp-rust.exe -i <input files> -o <output directory> 
```
For more options:
```
loft-cpp-rust.exe --help
```
